# 字符串与数字的运算
```php
if (isset($_GET['num'])) {
	$num = $_GET['num'];
	if (intval($num) < 2020 && intval($num + 1) > 2021) {
		echo("success");
	} else {
		die("failed");
	}
}
```
构造`num=1e5`即可
## 原理
在较低版本的`php`中，`intval`函数解析到字符串非数字部分直接结束，因而`'1e5'`被解析为`1`，
而当字符串同数字进行四则运算时会先行转换为数字，也就是`100000`，在进行相应的运算
